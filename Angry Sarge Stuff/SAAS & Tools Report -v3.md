## **1. The Video Infrastructure (The "Heavy Lift")**

### **Google Drive (The Warehouse)**

- **The Concept:** Our secure, off-site storage facility used **strictly for video project folders**. No clutter, just the heavy 4K files.
    
- **The "Fiber" Workflow:** Once the 1Gbps internet is live, we upgrade to the 2TB plan. Using the Google Drive desktop app, we treat the cloud like a virtual hard drive.
    
- **The Sync Trick:** When we work on a video, we mark the folder "Available Offline." It downloads for maximum editing speed. When finished, we switch it back to "Online Only." It wipes the files from our physical hard drives but keeps them safe in the cloud, saving our hardware from being buried under 15GB files.
    
- **Goal Growth:** By Shot Show 2027, you’ll have hundreds of hours of searchable, safe footage—not a pile of dying thumb drives.
    

### **Blackmagic Cloud (The Global Control Room)**

- **The Concept:** This is the "brain" of the edit. If Drive holds the "parts," Blackmagic Cloud holds the "instructions."
    
- **The Example:** I can start an edit in Indiana, and you can open that **exact same project** in Texas. We can both work in the same project at once. DaVinci won't "lose" the footage because it's synced via Drive.
    
- **Future Training Goal:** When you transition to training, I can be editing the "Instructional Video" in real-time while you’re still on the range filming the next drill.
